# RPG Manager

## Clonando o Repositório

Execute o comando abaixo para copiar os arquivos do GitHub para sua máquina utilizando o VSCode:

```bash
git clone https://github.com/PauloVictor21/rpg-manager.git
```

---

## Instalando as Dependências

Execute os comandos abaixo em sequência, um por vez:

```bash
cd rpg-manager
cd frontend
npm install

cd ..
cd backend
npm install
```

---

## Executando o Projeto

Abra **2 terminais** no VSCode.

### Terminal 1 — Backend

Execute os comandos abaixo:

```bash
cd backend
npm run dev
```

### Terminal 2 — Frontend

Execute os comandos abaixo:

```bash
cd frontend
npm run dev
```
