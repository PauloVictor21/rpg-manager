Código para copiar os arquivos do GitHub no VSCode:
git clone https://github.com/PauloVictor21/rpg-manager.git

Instalar dependências (execute em sequência, um por um):
cd rpg-manager
cd frontend
npm install

cd ..
cd backend
npm install


Agora, abra 2 terminais no VSCode;
Em um deles, coloque os seguintes códigos executando 1 de cada vez:
cd backend
npm run dev

Agora no outro terminal:
cd frontend 
npm run dev
